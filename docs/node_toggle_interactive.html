Each fires ≈3.5 µs (transport, H100/8B); each disabled-but-captured ≈0.18 µs (replay traversal). A real model has 2× #layers such nodes → see Results.

Core API

# once, around capture + after warmup
engine.enable_toggle_capture(True)        # record producer nodes during capture
engine.bind_graph_exec(graph, exec)        # register the captured exec

# at runtime, any time — flip the active set (no re-capture)
transport.set_active_hooks(enabled)       # cudaGraphNodeSetEnabled + meta gate, in lockstep

Lazy variant for high-frequency reconfigure: set_active_hooks_lazy() + per-graph ensure_graph_current() on replay.
